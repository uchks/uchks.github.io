OptiScaler Installation Guide

Important: Do not use this mod in online games! It could trigger anti-cheat software and lead to bans.

Install as non-nvngx (Enables all Features)

  1. Download OptiScaler:
    • Go to the Releases page and download the latest version
  2. Find the nvngx_dlss.dll file:

    Go to your game's installation folder and look for a file named nvngx_dlss.dll. It's typically located in:

    • For Unreal Engine games: <path-to-game>\Binaries\Win64\
    • Other games may store it in folders like plugins

    If the file isn't in the main folder, search for it elsewhere.

  3. Make a copy of nvngx_dlss.dll:
    • Copy the file to the same folder
    • Rename the copied file to nvngx.dll
  4. Prepare the OptiScaler mod:

    Download the OptiScaler mod and locate its nvngx.dll file. Rename the mod's nvngx.dll to one of these supported filenames (recommended: dxgi.dll):

    • dxgi.dll
    • winmm.dll
    • version.dll
    • wininet.dll
    • winhttp.dll
    • OptiScaler.asi (if using an ASI loader)
  5. Install the OptiScaler mod:

    Place the renamed OptiScaler DLL into your game's main executable folder (where the .exe file is located).

  6. For non-Nvidia GPUs:

    Check the GPU spoofing options for additional setup instructions.

Compatibility with Other Mods

If you use another mod like Reshade that also relies on dxgi.dll, you have two options:

  • Rename the original file to something like dxgi-original.dll OptiScaler will load it automatically.
  • Create a folder named plugins in the game directory and move the conflicting mod files there. OptiScaler will check this folder and load them as needed.

Important: Do not rename the nvngx.ini file! It must remain nvngx.ini.

Press the Insert Key to activate the Overlay Menu.

Install as nvngx.dll (Limited features)

Note: This installation method will disable Frame Generation and the Overlay Menu.

  1. Download OptiScaler:
    • Go to the Releases page and download the latest version
  2. Extract the files:
    • Unzip the downloaded archive
    • Copy the contents into your game's main executable folder
      (e.g. for Unreal Engine games) <path-to-game>\Binaries\Win64\
  3. Enable Signature Override:
    • Open the DlssOverrides folder
    • Run the file EnableSignatureOverride.reg and confirm when asked
  4. For non-Nvidia GPUs:
    • Follow the GPU spoofing options guide

Warning: This version may overwrite the libxess.dll file if it already exists in the game folder. Back up any important files first.
