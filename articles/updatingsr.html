Updating XeSS, DLSS, and FSR (3.1+)

Boost your gaming performance by replacing outdated DLLs for Intel XeSS, NVIDIA DLSS, and AMD FSR with their latest versions. This guide will walk you through the process for each upscaling technology.

1. Intel XeSS (Xe Super Sampling)

Download the latest XeSS SDK to get the most up-to-date DLL files for Intel's upscaling technology.

  1. Download the Latest XeSS SDK:
    • Intel XeSS SDK - DLLs are located in the 'bin' directory
    • Intel XeSS 2.0 SDK - Note: Will be removed once Intel updates their GitHub
  2. Replace DLL Files:

    Find the existing DLLs in your game folder and overwrite them:

    • libxess.dll - Super Resolution
    • libxess_fg.dll - Frame Generation (not used yet)
    • libxell.dll - Low Latency (not used yet)

2. NVIDIA DLSS (Deep Learning Super Sampling)

  1. Download the Latest DLSS DLLs:
    • DLSS Super Resolution
    • DLSS Frame Generation
    • DLSS Ray Reconstruction
  2. Replace DLL Files:

    Overwrite the existing DLLs in your game directory:

    • nvngx_dlss.dll - Super Resolution
    • nvngx_dlssg.dll - Frame Generation
    • nvngx_dlssd.dll - Ray Reconstruction

3. AMD FSR (FidelityFX Super Resolution)

Important: FSR 1, 2, or 3.0 cannot be updated to 3.1+. This guide applies only to 3.1 and newer versions.

  1. Download the Latest FSR SDK:
    • AMD FidelityFX SDK - DLLs are located in the 'bin' directory
  2. Replace DLL Files:

    Locate and replace the existing DLLs (if they exist):

    • amd_fidelityfx_dx12.dll - DirectX 12
    • amd_fidelityfx_vk.dll - Vulkan
